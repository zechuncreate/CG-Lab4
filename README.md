## 实验功能（必做+选做全完成）
### 一、必做任务
1. 构建三维场景：数学隐式定义红色球体（左侧）和紫色圆锥（右侧），坐标、颜色完全符合实验要求
2. 光线投射与深度测试：实现类似Z-buffer的深度竞争逻辑，保证物体遮挡关系正确
3. Phong光照实现：完整实现环境光（Ambient）、漫反射（Diffuse）、镜面高光（Specular）三分量计算
4. UI交互面板：通过Taichi UI提供4个滑动条，实时调节Ka、Kd、Ks、Shininess参数，直观观察光照效果变化

### 二、选做任务
1. Blinn-Phong模型升级：用半程向量H替换Phong模型的反射向量R，高光过渡更柔和，渲染效率更高，对比两者视觉差异
2. 硬阴影（Hard Shadow）：在物体交点处发射阴影射线，判定阴影区域，阴影内仅保留环境光，实现真实的光影遮挡效果

## 环境配置与运行方法
### 1. 依赖安装
本实验基于Taichi框架开发，需先安装Taichi依赖，打开终端运行以下命令：
pip install taichi

### 2. 运行步骤
1. 将本仓库中的代码文件（main.py）保存到本地文件夹
2. 打开终端，切换到代码所在文件夹
3. 运行命令：python main.py
4. 弹出渲染窗口，通过左侧参数面板调节光照参数，观察实时效果

## 核心参数说明
| 参数名称 | 作用 | 范围 | 默认值 |
| --- | --- | --- | --- |
| Ka（环境光系数） | 控制场景背景环境光强度，模拟多次反射的漫射光 | 0.0 ~ 1.0 | 0.2 |
| Kd（漫反射系数） | 控制物体表面漫反射强度，体现物体本身颜色亮度 | 0.0 ~ 1.0 | 0.7 |
| Ks（镜面高光系数） | 控制物体表面镜面高光的亮度 | 0.0 ~ 1.0 | 0.5 |
| Shininess（高光指数） | 控制高光区域的收敛程度，数值越大，高光越集中、物体越光滑；数值越小，高光越弥散、物体越粗糙 | 1.0 ~ 128.0 | 32.0 |

## 选做内容补充说明
### 1. Blinn-Phong与Phong模型差异
Phong模型通过计算反射向量R，高光区域较小、边缘较硬；Blinn-Phong模型通过计算半程向量H（H = normalize(L + V)），高光范围更大、过渡更柔和，在大入射角下视觉效果更自然，且渲染开销更低。

### 2. 硬阴影实现原理
在光线与物体的交点处，向光源方向发射一条阴影射线，若该射线在到达光源前与其他几何体相交，则判定该点处于阴影区域，此时仅计算环境光分量，漫反射和镜面高光分量置零，实现硬阴影效果。
